# Support

Orbix is community-supported pre-1.0 software.

## Before Opening an Issue

- Read [README.md](README.md) for local setup and verification commands.
- Read [deploy/README.md](deploy/README.md) for Portainer/NAS deployment, backups, hardware transcoding, and troubleshooting.
- Search existing issues and pull requests.
- Check whether the problem reproduces on the current development branch.

## Where to Ask

- Bugs: use the bug report issue template.
- Feature requests: use the feature request issue template.
- Setup or deployment questions: use the support question template.
- Vulnerabilities: follow [SECURITY.md](SECURITY.md), not public issues.

## Public Issue Safety

Do not include:

- Real `.env` or `deploy/.env` files.
- TMDB tokens, session secrets, database passwords, or API keys.
- Full private media paths if they reveal personal information.
- Logs that include cookies, authorization headers, or secrets.

## Expectations

There is no guaranteed response time. Clear reproduction steps, environment details, and logs with secrets removed make support much easier.
